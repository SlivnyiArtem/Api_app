Для вызова проекта заполните файл UserInput.Json в папке Api_App внутри этой папки.
В файле 4 поля:
ID - id кинопоиска
Name - предполагаемое имя
Type - тип запроса
SavePath - путь сохранения

Поля тип и путь сохранения обязательны для исполнения
В поле SavePath укажите путь к папке, куда сохранять файлы с информаций полученной по запросу(путь окончить на \\) Любой \ дублировать
В поле типа может быть указан один из следующих типов 
1) I - получить информацию о фильме
2) V - получить ссылки на трейлеры и другие видео о фильме
3) U - получить информацию о сиквелах и приквелах фильма, если они существуют
4) S - получить информацию о съёмочной группе фильма
5) G - получить информацию о фильмах похожего жанра(по мнению Кинопоиска)

Для получения значимого результата требуется одно из первых полей.
Поле id - точное id фильма на Кинопоиске. Если оно в наличии поле имени игнорируется(sig!)
поле Name - предполагаемое имя искомого фильма(поле будет рассматриваться, если не заполнено поле id). 
Кинопоиск вернет требуемую информацию по первым нескольким наиболее популярным фильмам, обладающих таким названием или похожим
